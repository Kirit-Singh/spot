## The bridge, and the one rule that makes it safe

A native Stage-2 ranking row is exactly `{target_id, arm_value, evaluable, rank}` — **no target
namespace, no perturbation modality**. Those two facts exist ONLY in W3's bridge. Without it
Stage 3 would have to infer identity from an id's string SHAPE (the universe holds 11,522 Ensembl
ids *and* 4 bare symbols — MTRNR2L1/4/8, OCLM, three of which carry an ENSG-looking release key
belonging to a DIFFERENT gene) and default a modality from a config constant — a *setting* wearing
the costume of an assay.

> **The bridge may ADD facts the native bytes lack (namespace, modality).
> It may NEVER CHANGE a fact the native bytes already state (arm_value, rank, evaluable).**

`arm_value` / `rank` / `evaluable` are therefore read from the **ADMITTED NATIVE ranking records**
and the bridge is required to agree. The drug direction is **re-derived** from that native value
(`SIGN_EPS = 1e-9`, Stage-2 Direct `config.py`); the serialized `desired_target_modulation` is a
*check*, never an input. **The RECEIPT is required** — W3's bridge report names the bytes it judged
but nothing ties it to an admitted aggregate, so a report without a receipt is an ADMIT that names
no release.

The emitted bundle **NAMES its bridge** — in `stage2_aggregate.stage3_bridge`, in `provenance`, and
in the **bundle id**. The same aggregate and store under a *different* admitted bridge must yield a
*different* bundle; otherwise the bundle would be detached from the one artifact its target
identities and its assay come from.

## The named refusals (each has a test that FIRES on a RE-SEALED forgery)

`druglink.stage2_bridge` — fail-closed, named gates. Every mutation below is re-sealed first,
because a forger with repo access recomputes a self-hash; the gate has to stop a *consistent*
forgery, not a careless one.

| attack | gate |
|---|---|
| bridge bytes edited after sealing | `the_bridge_does_not_recompute_its_own_bridge_sha256` |
| bridge binds nothing | `the_bridge_names_neither_the_bytes_nor_the_admission_it_stands_on` |
| report swapped / judged other bytes | `the_bridge_report_judged_a_different_bridge` |
| report did not admit | `the_bridge_verifier_did_not_admit_this_bridge` |
| **receipt ABSENT** | `the_release_ships_no_stage2_stage3_receipt` |
| receipt swapped (other bridge / other aggregate) | `the_receipt_binds_bytes_that_are_not_the_ones_on_disk` |
| a NATIVE arm value or rank changed | `a_bridge_row_restates_a_measurement_the_native_bytes_already_made` |
| identity namespace altered | `a_bridge_row_carries_no_usable_target_namespace` |
| modality changed or absent | `a_bridge_row_does_not_declare_what_the_assay_did` |
| modulation ≠ the re-derived sign | `the_serialized_modulation_disagrees_with_the_sign_re_derived_from_the_native_arm_value` |
| bridge over a different aggregate | `the_bridge_was_built_over_an_aggregate_stage3_did_not_admit` |
| orphan row | `a_bridge_row_the_admitted_native_bytes_do_not_produce` |
| dropped row | `the_native_bytes_produce_a_row_the_bridge_dropped` |
| duplicate row | `two_bridge_rows_claim_one_lane_arm_target_identity` |
| a typed row on the pathway lane | `the_pathway_lane_contributed_a_typed_target_row` |
| a pathway context carrying target evidence | `a_pathway_context_carries_target_evidence_fields` |
| a bundle that names no bridge | `the_bundle_would_not_name_the_bridge_that_typed_its_rows` |
| a bundle naming a DIFFERENT bridge | `the_bundle_does_not_name_the_bridge_that_typed_its_rows` *(verifier)* |

## Architecture held

- the public store is **global and selection-independent**; a selection is a **projection**
- **no combined candidate rank**, no combined objective, no p/q/FDR at any depth
- typed origins stay separate; **pathway is CONTEXT and never sources a drug edge**
- an untested inverse is **hypothesis-only** — queued for a look, never observed support
- v1 (`361d0833…`) is byte-identical and did not budge. The **v2** schema moved because the
  producer emits a new binding (`stage2_aggregate.stage3_bridge`), and the v2 / schema-set digests
  were re-pinned from the producer's own bytes — never typed by hand.
